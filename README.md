#LANGKAH-LANGKAH GIT REPO#

langkah 1:
- git init

langkah 2:
- git add [nama file yg ingin di commit]
- git add . (akan masukin semua file yg ada di folder ke keranjang)

masukkin ke keranjang untuk di commit

langkah 3:
- git status

liat apakah file yg ingin di commit sudah masuk ke keranjang, dan pasti sudah hijau

langkah 4:
- git commit -m 'nama commit'

langsung commit yg ada di keranjang

langkah 5:
remote repository
- login ke github
- bikin repository
- git remote add origin [nama repository githubnya]
- git branch -M [nama]
- git push -u origin [nama branch]
